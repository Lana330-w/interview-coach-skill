# interview-coach-skill
# 🎯 Interview Coach — Claude AI 模拟面试技能包

一个基于 Claude 的 AI 模拟面试工具，根据你的简历和目标岗位生成个性化面试题，对你的每个回答进行实时评分与反馈，最终生成一份完整的面试评估报告。

---

## ✨ 功能介绍

- 📄 **简历解析** — 支持上传 PDF / DOCX 文件，或直接粘贴简历文本
- 🎯 **个性化出题** — 根据你的真实经历 + 目标岗位 + JD 岗位描述生成 10–20 道定制面试题
- 💬 **实时反馈** — 每道题作答后立即给出评分和具体点评
- 📊 **面试报告** — 生成包含雷达图、逐题得分、优劣势分析、学习建议的完整评估报告
- 🌐 **中英双语** — 同时支持中文和英文面试场景

---

## 📦 安装方法

1. 前往 [Releases](../../releases) 页面下载最新的 `interview-coach.skill` 文件
2. 打开 [Claude.ai](https://claude.ai)
3. 进入 **Settings（设置）→ Skills**
4. 上传 `interview-coach.skill` 文件，安装完成

---

## 🚀 使用方法

安装后，在 Claude 对话框中直接说：

> "我想模拟面试"

或者：

> "帮我准备面试，我要面试软件测试工程师"

然后按提示提供以下信息：

| 信息 | 是否必填 | 说明 |
|---|---|---|
| 简历内容 | ✅ 必填 | 上传文件或粘贴文本均可 |
| 目标岗位 | ✅ 必填 | 如：软件测试工程师、前端开发、产品经理 |
| 目标年限/级别 | ✅ 必填 | 如：3–5年、Senior、初级 |
| 岗位 JD | 💡 推荐填写 | 提供 JD 后题目更贴近真实面试，不填也可继续 |

---

## 📋 面试流程

```
第一步：提交简历 + 岗位信息
       ↓
第二步：Claude 生成 10–20 道个性化面试题
       ↓
第三步：逐题作答，每题获得实时评分与反馈
       ↓
第四步：面试结束，生成完整评估报告
```

---

## 📊 评估报告包含

- **综合得分**（满分 100 分）
- **六维能力雷达图**：技术能力 / 问题解决 / 沟通表达 / 团队协作 / 自我驱动 / 岗位匹配度
- **逐题得分表**：每道题的得分和简评
- **核心优势**：3 条具体亮点
- **待提升方向**：3 条具体改进建议
- **推荐学习方向**：针对薄弱点的个性化学习建议
- **面试准备度评级**：🔴 暂不建议投递 / 🟡 还需加强 / 🟢 可以投递 / ⭐ 表现出色
